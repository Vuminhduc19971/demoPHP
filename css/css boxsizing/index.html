Box sizing

content-box ( width là 300 => with thực tế > 300 khi có padding và border

Chiều rộng thực tế = css width + padding left + padding right + border left + border right

Chiều cao thực tế = css height + padding top + padding bottom + border top + border bottom

border-box ( kiểm soát được chính xác kích cỡ ) 300 là 300

Chiều rộng thực tế = css width

Chiều cao thực tế = css height
