HTML Class ( Lớp,nhóm,đội )

Class được dịch là lớp , nhóm hay đội trong tiếng việt

Chỉ ra các phần tử có 1 điểm chung nào đó

Thuộc tính class được sử dụng để định nghĩa các kiểu style giống nhau với các thẻ có tên class giống nhau

Vì vậy các thẻ html với tên class giống nhau sẽ hiển thị với định dạng và phong cách giống nhau

Chúng ta có danh sách các thành phố với thẻ div

Các tỉnh miền bắc đặt class là north

Các tỉnh miền trung đặt class là center

Các tỉnh miền trung đặt class là south
